Lambdas
~~~~~~~~~~

Lambda expression in java is an anonymous (no name) method
that does not need to define the data type of input parameters 
and does not need to have a return type.

Lambda Expression is a short block of code
which takes in parameters
and returns a value

SYNTAX:
() ->
(parameter) -> expression;
parameter -> expression;
(parameter1, parameter2) -> expression
(parameter1, parameter2) -> {code block}
(parameter1, parameter2) -> { return value;};
